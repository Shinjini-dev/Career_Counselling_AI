# Career Counselling AI

## Overview

Career Counselling AI is a Django-based web application that helps users identify suitable career paths based on their skills, interests, and responses to assessment questions.

## Features

* User Information Collection
* Skill Assessment Quiz
* Career Recommendation System
* Interactive User Interface
* Career Guidance Dashboard

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## Installation

```bash
git clone https://github.com/Shinjini-dev/Career_Counselling_AI.git
cd Career_Counselling_AI
pip install -r requirements.txt
python manage.py runserver
```

## Project Structure

* Quiz Module
* Career Recommendation Module
* User Information Module
* Database Management
